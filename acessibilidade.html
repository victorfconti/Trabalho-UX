Deficiência visual

Muitas pessoas sofrem de transtornos visuais, de forma que criar um site acessível neste aspectivo é uma chave do sucesso para qualquer sistema.

Existem diversas de auxíliar essas pessoas, uma das mais simples é utilizar a tag alt para descrever o conteudo das imagens.

<img src="RUX.svg" alt="logo do site">

Outra forma é utilizar as tags corretas na construção de web sites, para facilitar os leitores de telas.

Além de utilizar cores de alto contraste na construção do sistema, para facilitar a leitura de pessoas com baixa visão.
